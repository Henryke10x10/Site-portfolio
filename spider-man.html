Spider-Man Hero

Projeto de front-end demonstrado no evento FrontPush
O projeto cria um página promocional de lançamento do jogo Spider-man Milles Morales. As tecnologias que se destacam nesse projeto são o SCSS, pré-processador css, e a biblioteca javascript GSAP, usada para o motion do site.
